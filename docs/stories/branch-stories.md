# 支线事件（编年史事件）

> 关联文档：`docs/character-relations.md` — 角色关联与支线角色表 | `docs/story.md` — 主线故事与流程 | `docs/town-settings.md` — 青山镇设定

---

游戏没有传统的主线/支线之分，而是由一系列**编年史事件**构成。每次开局随机选择该轮经历哪些事件，每个事件可能关联特定角色、奖励、或剧情分支。

当前共 **6 个主要支线事件**：

1. [归海楼比武大会](#1-归海楼比武大会)
2. [青山之巅·六绝比武](#2-青山之巅六绝比武)
3. [九朵桃花酒吧杀人事件](#3-九朵桃花酒吧杀人事件)
4. [喝酒结拜比轻功](#4-喝酒结拜比轻功)
5. [天工坊造武器](#5-天工坊造武器)

### 1. 归海楼比武大会

| 字段         | 内容                  |
| :----------- | :-------------------- |
| **事件 ID**  | `chronicle_guihailou` |
| **地点**     | 归海楼                |
| **节点数**   | 3                     |
| **奖励类型** | 招式（action）        |

玄门及其他门派前往归海楼切磋交流。**军师**与**博士**以"观摩"为名陪同前往，实为收集实战数据，用于研发战斗芯片。

#### 流程

1. **抵达归海楼** — 归海楼山门前人声鼎沸。你递上请帖，小厮领你入座。台上桑原正在与人切磋，刀光剑影。观众席一角，两个身着斗篷的人正在低声交谈——后来你才听说，那是军师和博士，以"观摩"为名收集战斗数据。
2. **观战切磋** — 几轮切磋下来，你对各家路数有了新的理解。归海楼掌门一刀亲自下场与一位老道表演，剑气纵横，满座喝彩。
3. **收获** — 有所感悟，获得奖励。

#### 关联

- **角色**：桑原（归海楼弟子）、一刀（归海楼掌门）、腊月（表演赛）、军师、博士
- **故事线**：`sect`（n14 切磋、n15 表演赛）、`xuanmen`（n15 重逢黑云·小树）

### 2. 青山之巅·六绝比武

| 字段         | 内容                             |
| :----------- | :------------------------------- |
| **事件 ID**  | `chronicle_six_duel`             |
| **地点**     | 青山之巅（镇上一座古戏楼的屋顶） |
| **节点数**   | 1                                |
| **奖励类型** | 招式（action）                   |

青山绝顶，六位绝世高手约定在此比武。

#### 六绝

| 称号   | 角色      | 身份                           |
| :----- | :-------- | :----------------------------- |
| **逸** | 龙女      | 隐世奇人，关联奇遇流（授功法） |
| **破** | 孙悟      | 天命·棍，支线核心角色          |
| **闪** | 西瓜      | 霸刀，默认决赛 Boss            |
| **观** | 过儿      | 隐世奇人，关联奇遇流（赠兵器） |
| **悟** | 寻香      | 小凤·暗器，事件结束后送奖励    |
| **韧** | 酒鬼·无志 | 隐世高人，九朵桃花常客         |

六绝虽打得惊天动地，但都收着范围——底下就是老街，砸下去会伤到平民。

**张烈**与**博士**二人同行前往观战——博士负责收集战斗数据，张烈负责护卫。

#### 流程

1. **绝顶观战** — 青山绝顶上，六道人影分立各方。你远远望着，大气都不敢出。观众席另一侧，军师、朵儿、博士三人也在。他们说是来收集数据的，但你总觉得他们的眼神不太对。
2. **领悟** — 战斗结束后，寻香朝你这边看了一眼，微微点头，然后转身离去。你有所领悟，获得奖励。

#### 关联

- **故事线**：`wanderer`（**主线·n14 与朵儿同往观战**，过儿/龙女为恩师）

### 3. 九朵桃花酒吧杀人事件

| 字段         | 内容                    |
| :----------- | :---------------------- |
| **事件 ID**  | `chronicle_bar_killing` |
| **地点**     | 九朵桃花酒吧（杏花街）  |
| **节点数**   | 2                       |
| **奖励类型** | 修炼点（points）        |

#### 背景

**九朵桃花酒吧**——开在杏花街上，街对面就是**宝字堂药铺**（飞虎·竹子坐堂行医之处）。名义上老板常年在外，由"女儿"朵儿看店，实则幕后老板是副会长，酒吧是组织的情报据点。

竹子泡的跌打酒在街坊里有些名气，朵儿在他那儿拿货放在吧台后面卖，每隔几天竹子送一次货，顺便收回空瓶。

**灵素·唐柔**是唐门旁支，十八岁才得自由身。出来后想学正经医术，在宝字堂已经待了一阵子——竹子教她认药、辨症、处理伤口。她日常就是看店、碾药、跟诊。

| 角色          | 身份                                          | 关联                                                 |
| :------------ | :-------------------------------------------- | :--------------------------------------------------- |
| **朵儿**      | 酒吧"老板女儿"                                | 实际为组织成员。酒吧幕后老板为副会长，是组织情报据点 |
| **酒鬼·无志** | 隐世高人                                      | 常客。在酒吧触发打抱不平→结拜事件的前置              |
| **来风**      | 江湖散人                                      | 常客。打抱不平被随机一组人看到后→喝酒结拜            |
| **风水·四娘** | —                                             | 常客。结拜组成员                                     |
| **浩然·潮生** | —                                             | 常客。结拜组成员                                     |
| **黑云·小树** | 玄门旁支·已出走                               | 常客。结拜组成员                                     |
| **飞虎·竹子** | 宝字堂药铺老板兼医师 / 特殊事件调查科兼职法医 | 街对面宝字堂药铺。定期送跌打酒到酒吧，与朵儿对账     |

#### 事发现场

军方卧底被朵儿**约到后院**，以**毒杀**方式清除。张烈在门口**望风**。尸体伪装成暴毙模样。

唐柔在柜台后面碾药，竹子从后院出来，抱了一箱新泡好的跌打酒，说了句"走，送货"。唐柔放下药臼跟出去。过街进九朵桃花，竹子把箱子搁吧台上跟朵儿对账。唐柔站在旁边打量店里的陈设。

此时事发。朵儿在后院完成了清除。

竹子放下账本走过去蹲下，翻了翻伤口，没说话。过了几秒，头也没回地说了句："去，把我柜子里第三格那瓶药酒拿来。"唐柔转身跑了出去。

#### 各线视角

| 故事线                  | 视角                 | 细节                                                                                                                                     |
| :---------------------- | :------------------- | :--------------------------------------------------------------------------------------------------------------------------------------- |
| **奇遇流（叶寻）**      | **全程目击**         | 准备了礼物想给朵儿惊喜，从后门潜入——轻功极好，未被任何人发现。目睹了后院发生的一切                                                       |
| **军旅退伍（陆斐）**    | 事后得知             | 老柴已查出组织与 2074 年林家灭门有关的间接证据，准备上报时被灭口。掠影后来告知陆斐                                                       |
| **血海深仇（林晚风）**  | **现场勘查（~n14）** | 以特殊事件调查科身份到场。奇岚已先到。在尸体衣物夹层中发现一叠被烧过的纸屑残片——勉强拼出"2074"、半个"木"字偏旁、以及"义"和"火"等零散字迹 |
| **天生道种 / 玄门子弟** | 听闻传闻             | 在街上听说九朵桃花出了事，不知细节                                                                                                       |

#### 流程

1. **传闻** — 你在街上听说九朵桃花酒吧昨晚出了事。有人被杀，凶手消失得无影无踪。酒吧照常营业，但吧台后面那排药酒瓶少了几瓶。
2. **线索** — 街坊们议论纷纷，但没有人说得清真相。你只打听到死者是个外来人，平时不怎么露面。据说朵儿那晚在店里，但第二天照常开门，像什么都没发生过。

#### 关联

- **角色**：朵儿（行凶）、张烈（望风）、军方卧底（死者）、副会长（幕后）、飞虎·竹子（验伤）、灵素·唐柔（目击）、奇岚（勘查）
- **故事线**：`wanderer`（主线目击）、`veteran`（掠影告知）、`feud`（现场勘查）

### 4. 喝酒结拜比轻功

| 字段         | 内容                                            |
| :----------- | :---------------------------------------------- |
| **事件 ID**  | `chronicle_sworn_brothers`                      |
| **地点**     | 九朵桃花酒吧                                    |
| **节点数**   | 1                                               |
| **奖励类型** | 被动（passive）— 随机轻功 + 随机 `jiu` tag 功法 |

#### 前置

主角在**九朵桃花酒吧**遇到以下一组人——酒鬼（老人）、来风（中年）、风水、浩然、黑云（小叔）。

来风在酒吧附近打抱不平，被这组人中随机一人看到。随后一起喝酒，结拜为兄弟。

#### 流程

1. **九朵桃花** — 你在九朵桃花酒吧独酌。隔壁桌坐着几个人——一个老人、一个中年人和几个看起来也是江湖中人的男女。中年人刚才在街口打抱不平，被这几人拉进来喝酒。他们聊得热闹，见你一个人，便招呼你过去坐。酒过三巡，有人提议结拜。
2. **结拜** — 大家喝到兴起，轮番表演轻功助兴。你也露了一手，赢得一片叫好。最后几个人醉醺醺地拜了把子，约好以后有难同当。
3. **结拜之礼** — 获得奖励。

#### 关联角色

酒鬼、来风、风水、浩然、黑云

### 5. 天工坊造武器

| 字段         | 内容              |
| :----------- | :---------------- |
| **事件 ID**  | `tiangong_weapon` |
| **地点**     | 天工坊            |
| **节点数**   | 1                 |
| **奖励类型** | 武器（weapon）    |

#### 背景

天工坊主人**千星**，神铁、千机、阿赖耶识的锻造者。以炁驱动的电磁锤纵横，雷火交加。

每人一生只能给他下一个订单，做不做全凭他意。

#### 流程

1. **天工坊** — 斗炁大会即将开始，你在街上看到了天工坊的招牌。千星正靠在门口擦一把新出炉的兵器，看到你便扬了扬下巴：「哟，来了？这次进了决赛圈，要不要换件趁手的家伙？」
    - **选项 A**：去天工坊看看 → 挑选材料，获得武器奖励
    - **选项 B**：不去，在家修炼 → 潜心修炼（+4 修炼点）

### 6. 斗炁图书馆学功法

| 字段         | 内容                            |
| :----------- | :------------------------------ |
| **事件 ID**  | `douqi_library`                 |
| **地点**     | 斗炁图书馆（街角）              |
| **节点数**   | 1                               |
| **奖励类型** | 被动（passive）— 防御或攻击功法 |

#### 背景

位于街角的一座古朴图书馆，檐下匾额写着「斗炁图书馆」四个字。

- **管理员龙语仙** — 龙角少女，擅长防御功法
- **常客白狐·南宫** — 冰霜双刀使，埋头翻阅攻击功法古籍

#### 流程

1. **斗炁图书馆** — 你推开木门，一股书卷气扑面而来。柜台后一名龙角少女正悠闲地翻着书，见你进来便抬眼一笑："新面孔嘛，随便看。"

    角落的蒲团上，一只白狐蜷着尾巴，专心致志地盯着一本泛黄的古籍，尾巴尖时不时抖一下，完全没注意到你。
    - **找龙语仙请教防御功法** → 获得防御型被动
    - **找白狐请教攻击功法** → 获得攻击型被动
    - **随便翻翻就走** → 无奖励
