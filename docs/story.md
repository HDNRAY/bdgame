## 一、完整故事文案

---

### 楔子

1988年冬，青山镇东南深山。

炼炁协会会长——一个名字早已被时间磨去的老者——站在山腹深处。他面前悬浮着一块半透明的、不规则的晶体。

他花了二十五年，才找到打开它面前那扇“门”的方法。方法是错的。或者说，是对的，但代价超出了他的预期。

他触碰了“东西”。

那一瞬间，他看到了时间。不是一条线，而是一团乱麻——无数条时间线在同一个位置上互相缠绕，每一根线都通往一个不同的“青山镇”。每一个青山镇里都有一个他，有的死了，有的活着，有的从未来回来，有的等着去未来。

他看到的东西，让人想要回去。

然后他就“回去”了——整具身体消失在1988年冬的空气中。“东西”面前空无一人。会长失踪。

青山镇失去了会长，但炁道未绝。协会长老会以“代管”之名维持运转，等待下一位被“东西”选中的人出现。

这一等，就是八十六年。

---

### 2074年初 · 青山镇

这一年，青山镇有二十三个孩子同时出生。

其中一个，就是这一轮的主角。

他/她睁开眼看到的第一样东西，是病房天花板上老旧的日光灯。第二样东西，是一张脸——某个人的脸。微笑，或者疲惫，或者二者都有。

这个人是主角的“重要之人”。发小，兄弟姐妹，情人，或一个注定会在十八年后改变主角命运的人。

青山镇的一切看上去都很普通：有学校，有商店，有开了四十年的豆浆铺子。镇口有公交站，坐四十分钟能到高铁站，再坐两小时能到任何一座大城市。

唯一的异常是：全镇六千多居民，全都会“炁”。

小孩们白天学矩阵运算和义体维护，晚上回家站桩、吐纳、画符。两种知识在同一个大脑里并行不悖，没人觉得哪里不对。

主角在这样的环境里长大。六岁习炁，十岁通脉，十四岁能驭物/挥剑/断铁/画咒——根据流派背景不同而异。而“重要之人”始终在身边，像一棵树的另一根枝干，从同一个根上长出来，越靠越近。

日子像一条不会转弯的河，一直流到2088年。

---

### 2088年秋 · 斗炁大会

钟声。

不是从钟楼传来的，是从地底下传来的。从每一块青石板里，从每一面墙上，从每一把老旧的木椅的裂缝里——像是什么东西在青山镇的骨头上敲了一下。

所有人同时听到了：

> **“炁道不灭，薪火不孤。吾辈传人，当以斗炁决之。”**

斗炁大会启动了。规则直接刻入了青山镇的地脉，无人能改——海选产生三十二强，三十二进十六小组赛，十六人单败淘汰，最终胜者为下任会长，有权进入山腹。

原因？镇上最老的人也只记得一句口头相传的话：“会长没了，就斗炁。”

没有人知道，这个“钟声”的底层机制，与八十多年前失踪的那位会长有关。更没有人知道，写下这道祖训的人，来自更远的未来。

---

### 海选

主角决定参加。

修炼，构筑BD，适应义体与炁的共振——游戏前期的所有准备都指向一件事：通过海选。

海选的最后一关是“守门人”——镇上的老前辈，剑碑、铁壁、影婆之一。击败他/她，即可进入三十二强。

此时主角还不知道：

- 三十二强里有“拾遗”派来的精锐。
- 重要之人的命运，已经被大会的齿轮咬住了。
- 山腹里那个叫“东西”的玩意儿，会把所有人的人生碾成粉末。

---

### 三十二强 → 十六强

小组赛。四人一组循环对战，积分前二晋级。

主角在这一阶段：

- 遭遇“拾遗”伪装者——招式狠辣，与普通炼炁士的切磋完全不同。
- 遭遇童年旧识——或者为敌，或者为友。
- 遭遇场外暗手——有人在赛程表上做手脚，有人在饮用水里下东西，有人在观众席上等着决赛落幕。

更重要的是：**重要之人的命运线在这一阶段开始收紧。**

他/她可能出现在：

- 擂台的对面——作为必须击败的对手；
- 观众席上——作为被挟持的人质；
- “拾遗”的名单上——作为被选中的棋子；
- 某个深夜的巷子里——作为主角推开那扇门时看到的最后一样东西。

无论如何，主角必须继续赢下去。只有走到最后，才能触碰“东西”；只有触碰“东西”，才有机会改变这一轮命运。

---

### 淘汰赛 · 十六强 → 决赛

每一场都是生死局。对手越来越强，场外越来越暗。

决赛。

主角站在擂台上，对面是最后的对手——可能是重要之人，可能是“拾遗”的死士，可能是一个从未见过但战力极高的陌生人。

击败对方。成为会长。

---

### 山腹

主角推开最后一扇门。

路很长。两侧的石壁上刻着历代会长的名字，最近的几个名字还清晰可辨。再往前，字迹开始模糊。最远处有几个名字已经完全被风化吞没，只剩下凹陷的笔画轮廓。

路的尽头，“东西”悬浮在半空。一块半透明的、不规则的晶体，内部有极慢的光在流转。安静。不发光。不解释。

但“东西”前面站着一个人。

---

### 隐藏boss · 百年执念

那个人站在那里。或者说，那具躯体站在那里。

身形很高，但佝偻。全身上下几乎没有露出活人皮肤的缝隙——手臂是义体，脊柱是义体，胸腔外壳是义体，面部的下半张脸被金属覆盖，只有一双眼睛还保持着某种说不清的神色。

他/她已经一百岁了。

1989年初，他/她体穿至此处。为了活到下一次斗炁大会开启，他/她开始更换义体。十年。三十年。五十年。八十年。每换一次义体，就失去一部分记忆。到后来，连自己叫什么名字都不记得了。

只记得一件事：**要拿到“东西”。**

因为“东西”是唯一能让他/她“回去”的东西。虽然已经不记得要回哪里去了，但那种“想回去”的感觉像一根生锈的铁钉，钉在残存的意识最深处。

他/她看着主角，开口。嗓子里发出的声音像是金属片在砂纸上刮擦。可能说的是“来了？”可能说的是“你也想要？”也可能只是嘴唇动了动，什么都没发出来。

然后他/她出手了。

招式是上一轮玩家设计的BD。流派、义体配置、功法序列——全部都是上一轮玩家的构筑，被他/她用了一百年，磨成了本能。每一招都在预判，因为每一招都来自“曾经也是同样走到这里的人”。

---

### 击败

主角赢了。

他/她倒下的时候，义体外壳裂开，露出下面一层又一层的老旧义体骨架——那些是最早换上的部件，已经锈蚀发黑，缠绕着干涸的炁痕。

在彻底停止之前，他/她的眼睛忽然亮了一瞬。

那一点光里，像是有什么东西从极深的地方浮了上来。他/她看着主角，嘴唇动了动。

没有声音。

然后彻底停止。那具百年的、被义体反复掏空的身体，终于不再动了。

---

### “东西”面前

主角跨过那具残骸，走到“东西”面前。

没有任何声音。没有任何提示。但主角知道，“东西”在等一个选择。

选择只有两个：

**一、触碰“东西”——回到过去。** 成为下一个百年的拦路者。

**二、转身，离开。** 接受这一轮的失去，带着所有遗憾向前走，不再回头。

---

### 真结局

主角站在“东西”面前。

没有触碰它。没有回头。沿着来路，一步一步走出山腹。脚步声在黑暗的石道里回荡了很久。很久。

走出洞口的时候，天已经亮了。青山镇的炊烟正在升起。远处有人喊了一声什么——可能是邻居在打招呼，可能是某个孩子在叫谁的名字。

主角没有停。也没有回头。一直走出镇口，走上那条通往公交站的公路。

画面定格。

## 三、要素列表

---

### 世界观要素

| 要素         | 说明                                                         |
| :----------- | :----------------------------------------------------------- |
| **2088年**   | 故事发生的时间背景。赛博朋克世界观，义体改造普及             |
| **青山镇**   | 故事发生的舞台。地图上有名的普通小镇，实际为炼炁士后裔聚居地 |
| **炼炁士**   | 传承千年的修炼者群体。分为剑修、锻体、御物、诡术等流派       |
| **灵核芯片** | 每人出生时强制植入。记录生命体征，触发终末协议（禁区/死亡）  |
| **义体**     | 延长寿命/增强战力的手段。代价：长期使用会导致记忆与理智流失  |
| **“东西”**   | 封藏在山腹中的不明物体。唯一功能：让人“体穿”回过去           |
| **“拾遗”**   | 暗处组织，目标是夺取“东西”。名字只出现在碎片文本中           |
| **斗炁大会** | 会长之位空缺时的继承机制。海选→32强→小组赛→16强淘汰          |

---

### 角色系统

| 要素           | 说明                                                      |
| :------------- | :-------------------------------------------------------- |
| **每轮主角**   | 每一局游戏都是不同的人。不同背景、不同重要之人            |
| **流派背景**   | 剑修/锻体/御物/诡术。决定初始BD和成长剧情走向             |
| **重要之人**   | 每轮随机生成。关系、身份、命运锚点均为该轮独有            |
| **隐藏boss**   | 上一轮选择了“回去”的主角。体穿到1989年，100岁，全身义体化 |
| **守门人**     | 海选最终关。32个预设计NPC中模拟战力最低者（设计层）       |
| **“拾遗”成员** | 伪装成普通参赛者的暗桩。身份在碎片中逐步揭示              |

---

### 核心机制

| 要素              | 说明                                                     |
| :---------------- | :------------------------------------------------------- |
| **体穿**          | 触碰“东西”后整个人回到过去。落点固定：1989年初           |
| **轮回**          | 每轮是不同的时间线/不同的人。故事轮回，人不轮回          |
| **碎片系统**      | 游戏过程中收集的线索文字/画面。揭示旧轮回的痕迹          |
| **BD构筑**        | 流派+义体+功法搭配，每轮可不同                           |
| **单挑肉鸽**      | 核心玩法。每一轮对手配置随机，死亡/失败后重新开局        |
| **义体-记忆对冲** | 换义体延寿→战力提升，但碎片收集上限降低/部分线索永久关闭 |

---

### 流程结构

| 阶段     | 内容                             | 时长（剧情时间）  |
| :------- | :------------------------------- | :---------------- |
| **成长** | 童年经历、结识重要之人、发现痕迹 | 2074-2088（14年） |
| **准备** | 修炼、构筑BD、海选、击败守门人   | 2088秋（数月）    |
| **大会** | 小组赛+淘汰赛，遭遇各类对手/事件 | 2088年底（数周）  |
| **终局** | 击败隐藏boss，面对“东西”做出选择 | 即时              |

---

### 结局设计

| 结局         | 条件                   | 内容                                 |
| :----------- | :--------------------- | :----------------------------------- |
| **循环结局** | 触碰“东西”             | 体穿至1989年初，成为下一轮的隐藏boss |
| **真结局**   | 转身离开               | 不再回头，带着遗憾继续活下去         |
| **中途死亡** | 战斗中败北             | 直接结算，可重新开局（新主角/新线）  |
| **BAD END**  | 重要之人死亡且主角放弃 | 剧情结算，可重新开局                 |

---

### 隐藏要素

| 要素               | 说明                                                |
| :----------------- | :-------------------------------------------------- |
| **旧痕迹**         | 每轮可发现的上一轮/上上轮主角留下的刻字、遗物、笔记 |
| **“拾遗”真名**     | 只出现在碎片中，不会在主线直接揭示                  |
| **1989年失踪会长** | 上一轮隐藏boss留下的历史痕迹                        |
| **钟声来源**       | 碎片中可拼出：钟声由“未来者”与上届会长共同开发      |
| **“东西”来源**     | 从未在任何碎片中完整说明——始终保持不可知            |
