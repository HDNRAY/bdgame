## 一、完整故事文案

---

### 2074年初 · 青山镇

公元2074年。

义体改造已经是日常的一部分。街边的诊所和眼镜店一样多，换一条手臂和配一副隐形眼镜差不多快。全息广告牌在城市上空轮流播放，内容每天换，但看起来都差不多。人们走在街上，脚步快慢不一，机械关节的声音和肉身关节的声音混在一起，谁也听不出区别。

每个人出生时都会被植入一枚灵核芯片，比指甲盖还小。它平时什么也不做，只是安静地待在那里，只在一件事发生时触发相关部门处理：**被带入禁区**。

禁区在地图上有标注，但不会写明原因。普通人不会去，也不需要知道为什么。

赛博都市的霓虹覆盖了几百公里。但往南走，高铁四十分钟后，有一站叫青山镇。镇子不大，有街有巷，有开了几十年的铺子，也有近年新装的门头。镇口有一台自动售货机，玻璃上有一道裂纹，里面有几瓶落灰的饮料。

青山镇看上去和任何一座南方小镇没什么不同。只是这里的居民，从很久以前开始，就会一些“别的”东西——吞吐、凝练、运转。他们没有把它们叫做“炁”，但那就是炁。这些东西没有被写在课本里，也没有被记录在官方档案上，但每一代人都传了下来，像一种不需要解释的习惯。

没有人知道这座镇的秘密。也没有人知道镇东南方向的山里有什么。知道的人没有机会说，不知道的人也不会问。

2074年，一切都和往年一样，安静地运转着。

像是有什么东西在等。像是还没有到该响的时候。

---

### 2088年秋 · 斗炁大会

世间总有觊觎者。

有人一直盯着青山镇山腹里的那样东西。但青山镇在官方和炼炁协会的双重保护之下，强攻无异于自毁。

于是他们选了更慢的路——渗透。从外围开始，一层一层往里走，像水渗进墙缝。很久了。久到镇上的人已经忘了那些面孔是什么时候出现的，只记得他们一直都在。上届会长在位太久，久到足够一张网从最外层织到最核心。

会长的最后几年，身边照顾他的人，就是这张网最中心的那一根线。端药、传话、记录、陪伴——所有会长对外传递的信息，都经过这双手。

会长没有立遗嘱。或者说，他立过，但从未被送出去。

会长死后，照顾他的人开始着手接管。只差最后几步。

但会长死后第七日，一道震鸣从会长的炁印中扩散开来，穿透墙壁、地脉、砖石，抵达青山镇每一个居民的耳中：

> **“炁道不灭，薪火不孤。吾辈传人，当以斗炁决之。”**

斗炁大会强制启动。规则被刻入青山镇的地脉：全镇范围内，不分身份，不分流派，以单挑战的方式逐出下任会长。海选，三十二强，小组赛，十六人淘汰。

暗处的人措手不及。但箭已在弦上，只能将精锐伪装成参赛者，混入大会。

### 海选

主角决定参加。

修炼，构筑BD，适应义体与炁的共振——游戏前期的所有准备都指向一件事：通过海选。

海选的最后一关是“守门人”——镇上的老前辈，剑碑、铁壁、影婆之一。击败他/她，即可进入三十二强。

此时主角还不知道：

- 三十二强里有“拾遗”派来的精锐。
- 重要之人的命运，已经被大会的齿轮咬住了。
- 山腹里那个叫“东西”的玩意儿，会把所有人的人生碾成粉末。

---

### 三十二强 → 十六强

小组赛。四人一组循环对战，积分前二晋级。

主角在这一阶段：

- 遭遇“拾遗”伪装者——招式狠辣，与普通炼炁士的切磋完全不同。
- 遭遇童年旧识——或者为敌，或者为友。
- 遭遇场外暗手——有人在赛程表上做手脚，有人在饮用水里下东西，有人在观众席上等着决赛落幕。

更重要的是：**重要之人的命运线在这一阶段开始收紧。**

他/她可能出现在：

- 擂台的对面——作为必须击败的对手；
- 观众席上——作为被挟持的人质；
- “拾遗”的名单上——作为被选中的棋子；
- 某个深夜的巷子里——作为主角推开那扇门时看到的最后一样东西。

无论如何，主角必须继续赢下去。只有走到最后，才能触碰“东西”；只有触碰“东西”，才有机会改变这一轮命运。

---

### 淘汰赛 · 十六强 → 决赛

每一场都是生死局。对手越来越强，场外越来越暗。

决赛。

主角站在擂台上，对面是最后的对手——可能是重要之人，可能是“拾遗”的死士，可能是一个从未见过但战力极高的陌生人。

击败对方。成为会长。

---

### 山腹

主角推开最后一扇门。

路很长。两侧的石壁上刻着历代会长的名字，最近的几个名字还清晰可辨。再往前，字迹开始模糊。最远处有几个名字已经完全被风化吞没，只剩下凹陷的笔画轮廓。

路的尽头，“东西”悬浮在半空。一块半透明的、不规则的晶体，内部有极慢的光在流转。安静。不发光。不解释。

但“东西”前面站着一个人。

---

### 隐藏boss · 斗炁协会副会长（地下组织：义体研究部）

那个人站在那里。或者说，那具躯体站在那里。

身形很高，但佝偻。全身上下几乎没有露出活人皮肤的缝隙——手臂是义体，脊柱是义体，胸腔外壳是义体，面部的下半张脸被金属覆盖，只有一双眼睛还保持着某种说不清的神色。

他/她**斗炁协会副会长**，已经一百二十岁了。

1989年初，他/她体穿至此。他/她知道"东西"在山腹里，也触碰过它一次。那时他/她还很年轻，刚刚赢了斗炁大会，成为会长。

他/她站在"东西"面前时，听到的是一个声音——不是在耳朵里响起的，而是在脑子里直接出现的。那个声音说：**我可以实现你一个愿望。一个作用于你自身的愿望。变强、长高、富有……桀桀桀桀——**

他/她选了回到过去，见那些已经不在的人。

"东西"说"好呀~"。然后他/她就出现在了1989年初。

体穿后的几十年里，青山镇的会长换了一任：老会长去世，儿子继位。那时他/她才意识到，想再面对"东西"就必须成为会长。但他/她的实力还远远不够。他/她选择了一条更安静的路——辅佐年轻的会长，帮他站稳脚跟，成为他最信任的人。

2020年左右，他/她五十一岁。两个知道"东西"存在的人坐下来，秘密留下了一道后手：钟声。会长死后强制触发斗炁大会，用比武决定谁有资格面对"东西"。他们约好，这件事对谁也不说。

从那之后，他/她开始做准备——培养人。因为他/她知道，钟声响起时自己已经太老了，需要有够强的人替他去面对"东西"。

但为了保持战斗能力，副会长开始加装义体。每换一次，就失去一部分记忆。

先忘的是小事。然后是钟声的细节。然后是培养这批人的初衷。到后来，他/她只记得一件事：**要拿到"东西"。**

虽然已经不记得为什么要拿了。但那种"必须拿到"的感觉，像一根生锈的铁钉，钉在残存的意识最深处。

培育出的人一代传一代。这些人在副会长神志清醒时留下了足够的信息——哪些人是绊脚石、未来会发生什么大事。组织就是在他/她授意下成立的：**斗炁协会义体研究部**（地下名待定），表面研究义体技术，实则为夺取"东西"铺路。第一批（张烈那一代）到2088年已经四十多岁。第二批（阿九那一代）二十出头。

#### 博士

义体研究部的核心人物。坐轮椅，全身义体化——除了大脑以外几乎全部替换过。专研义体开发，擅长暗器。性格冷静，与军师搭档行动。他会出现在多个编年史事件中，以"收集战斗数据"为名介入。

#### 军师

副会长失智后，组织中实际管事的人——一个五十多岁的义体御物师，没人知道他/她的全名，只叫"军师"。

上届会长一直活到2088年。他守住了秘密，没有对任何人说过山腹里有什么。这是他和隐藏boss七十年前的约定，最后一部分。

现在钟声响了。但他/她已经不太记得钟声是什么意思了。

只记得有人会来。

他/她看着主角，开口。嗓子里发出的声音像是金属片在砂纸上刮擦。可能说的是"来了？"可能说的是"你也想要？"也可能只是嘴唇动了动，什么都没发出来。

然后他/她出手了。

---

### 击败

主角赢了。

他/她倒下的时候，义体外壳裂开，露出下面一层又一层的老旧义体骨架——那些是最早换上的部件，已经锈蚀发黑，缠绕着干涸的炁痕。

在彻底停止之前，他/她的眼睛忽然亮了一瞬。

那一点光里，像是有什么东西从极深的地方浮了上来。他/她看着主角，嘴唇动了动。

没有声音。但口型大概是三个字。

**别信它。**

然后彻底停止。那具百年的、被义体反复掏空的身体，终于不再动了。

---

### “东西”面前

主角跨过那具残骸，走到“东西”面前。

没有光。没有声音。但有一个声音直接在脑子里响起来——不是语言，更像是一种意识直接灌入：

**我可以实现你一个愿望。一个作用于你自身的愿望。变强、长高、富有……桀桀桀桀——**

主角不知道的是：上一任会长的遗嘱——关于"东西"和钟声的全部真相——被"军师"扣在手里，从未送到继任者面前。

没有人告诉过主角这个东西会说话。也没有人告诉过主角，它说的话不能全信。

它只是说可以实现你的愿望。你说了，它做了，然后沉默。至于它做的和你说的，是不是同一件事——等你发现的时候，已经来不及了。

历届来到这里的会长都说了同一个愿望。

现在轮到你了。

选择只有两个：

**一、许愿——"我想再见见那些已经不在的人。"** "东西"说"好呀~"。下一秒，你出现在1989年初。青山镇的样子和一百年前没什么不同。你认识的人一个都不在这里。你要等。等一百年。等斗炁大会重新开启。等你自己重新走到这里。

**二、转身，离开。** 不去信它说的话。接受这一轮的失去，带着所有遗憾向前走，不再回头。

---

### 真结局

主角站在“东西”面前。

没有许愿。没有回头。沿着来路，一步一步走出山腹。脚步声在黑暗的石道里回荡了很久。很久。

走出洞口的时候，天已经亮了。青山镇的炊烟正在升起。远处有人喊了一声什么——可能是邻居在打招呼，可能是某个孩子在叫谁的名字。

主角没有停。也没有回头。一直走出镇口，走上那条通往公交站的公路。

画面定格。

## 三、编年史事件

> 游戏没有传统的主线/支线之分，而是由一系列**编年史事件**构成。每次开局随机选择该轮经历哪些事件，每个事件可能关联特定角色、奖励、或剧情分支。

### 1. 归海楼比武大会

玄门及其他门派前往归海楼切磋交流。**军师**与**博士**以"观摩"为名陪同前往，实为收集实战数据，用于研发战斗芯片。

- 关联角色：桑原（归海楼弟子）、一刀（归海楼掌门）、腊月（表演赛）、玄门线主角
- 关联故事：`sect`（n14 切磋、n15 表演赛）、`xuanmen`（n15 重逢黑云·小树）

### 1.5 河畔古庙（公共支线）

任一故事线都可能触发。主角在河边修炼时偶遇一座古庙，庙中隐世老者**酒鬼·无志**邀你切磋。切磋后的选择会影响后续喝酒结拜事件的解锁。

- 关联角色：酒鬼·无志
- 前置：无（公共支线，随机触发）
- 后继：若选择得当，解锁编年史事件 3（喝酒结拜）

### 2. 青山之巅·六绝比武

青山绝顶，六位绝世高手约定比武：**逸**龙女、**破**孙悟、**闪**西瓜、**观**过儿、**悟**寻香，以及尚未现身的**韧**。军师、朵儿、博士三人同行前往，表面收集战斗数据，实则企图制造意外伤亡。

- 寻香在事件结束后会赠送奖励
- 关联故事：`wanderer`（过儿/龙女为奇遇流恩师）

### 3. 喝酒结拜

风水（四娘）、浩然（潮生）、何云（黑云·小树）、无志（酒鬼）四人喝酒结拜。事件结束后获赠酒类奖励。

- 前置：需要先经历河畔古庙公共支线，与酒鬼·无志切磋并做出选择，才能解锁此结拜事件
- 奖励：按关系深浅赠送不同酒类物品

### 4. 组织杀军方卧底

义体研究部发现有军方卧底渗透。**朵儿**奉命执行清除——她用了某种让人查不到的手段绕过芯片监控完成击杀。

- 关联故事：`veteran`（兄弟就是在这次卧底行动中被害）、`wanderer`（主角可能目睹朵儿执行任务）

### 5. 军旅线·阶段1选择

军旅线专属分支。在部队初期面临选择：

- **A. 接受卧底任务**：被派往组织卧底，途中目睹队友被杀，为继续隐藏身份而加装义体。
- **B. 正常退伍**：按原设计走完军旅线，不涉及组织渗透。

## 四、要素列表

---

### 世界观要素

| 要素           | 说明                                                                 |
| :------------- | :------------------------------------------------------------------- |
| **2088年**     | 故事发生的时间背景。赛博朋克世界观，义体改造普及                     |
| **青山镇**     | 故事发生的舞台。地图上有名的普通小镇，实际为炼炁士后裔聚居地         |
| **炼炁士**     | 传承千年的修炼者群体。分为剑修、锻体、御物、诡术等流派               |
| **灵核芯片**   | 每人出生时强制植入。记录生命体征，进入禁区时触发相关部门处理         |
| **义体**       | 延长寿命/增强战力的手段。代价：长期使用会导致记忆与理智流失          |
| **“东西”**     | 封藏在山腹中的不明物体。唯一功能：让人"体穿"回过去                   |
| **义体研究部** | 斗炁协会下属的地下组织，表面研究义体技术，实为隐藏boss培养的夺宝势力 |
| **斗炁大会**   | 会长之位空缺时的继承机制。海选→32强→小组赛→16强淘汰                  |

---

### 角色系统

| 要素           | 说明                                                                             |
| :------------- | :------------------------------------------------------------------------------- |
| **每轮主角**   | 每一局游戏都是不同的人。不同背景、不同重要之人                                   |
| **流派背景**   | 剑修/锻体/御物/诡术。决定初始BD和成长剧情走向                                    |
| **重要之人**   | 每轮随机生成。关系、身份、命运锚点均为该轮独有                                   |
| **隐藏boss**   | 上一轮选择了"回去"的主角。斗炁协会副会长，体穿到1989年，全身义体化，记忆几乎清零 |
| **博士**       | 义体研究部核心。坐轮椅，全身义体化（除大脑），专研义体开发，擅长暗器             |
| **守门人**     | 海选最终关。32个预设计NPC中模拟战力最低者（设计层）                              |
| **“拾遗”成员** | 伪装成普通参赛者的暗桩。身份在碎片中逐步揭示                                     |

---

### 核心机制

| 要素         | 说明                                              |
| :----------- | :------------------------------------------------ |
| **体穿**     | 向"东西"许愿后整个人回到过去。落点固定：1989年初  |
| **轮回**     | 每轮是不同的时间线/不同的人。故事轮回，人不轮回   |
| **碎片系统** | 游戏过程中收集的线索文字/画面。揭示旧轮回的痕迹   |
| **BD构筑**   | 流派+义体+功法搭配，每轮可不同                    |
| **单挑肉鸽** | 核心玩法。每一轮对手配置随机，死亡/失败后重新开局 |

---

### 流程结构

| 阶段 | 内容 | 时长（剧情时间） |
| :------- | :------------------------------- | :----------------------------------------- || **编年史** | 随机选择该轮经历的事件组合（归海楼/六绝/结拜/卧底等） | 穿插在准备与大会之间 || **成长** | 童年经历、结识重要之人、发现痕迹 | 2074-2088（14年） |
| **准备** | 修炼、构筑BD、海选、击败守门人 | 2088秋（数月） |
| **大会** | 小组赛+淘汰赛，遭遇各类对手/事件 | 2088年底（数周） |
| **终局** | 击败隐藏boss，面对“东西”做出选择 | 战斗在2088年最后时刻，面对东西已经是2089了 |

---

### 结局设计

| 结局         | 条件           | 内容                                       |
| :----------- | :------------- | :----------------------------------------- |
| **循环结局** | 许愿"回到过去" | 被"东西"送回1989年初，成为下一轮的隐藏boss |
| **真结局**   | 转身离开       | 不再回头，带着遗憾继续活下去               |
| **中途死亡** | 战斗中败北     | 直接结算，可重新开局（新主角/新线）        |

---

### 隐藏要素

| 要素           | 说明                                                         |
| :------------- | :----------------------------------------------------------- | --- |
| **旧痕迹**     | 每轮可发现的上一轮/上上轮主角留下的刻字、遗物、笔记          |
| **“拾遗”真名** | 只出现在碎片中，不会在主线直接揭示                           |     |
| **钟声**       | 会长死后强制启动斗炁大会的机制。由隐藏boss与上届会长共同留下 |
| **“东西”来源** | 从未在任何碎片中完整说明——始终保持不可知                     |
