# "东西"设定

> 关联文档：`docs/story.md` — 世界观要素、终局篇
> `docs/town-settings.md` — 青山镇地理环境

---

## 概述

"东西"是封藏在青山镇东南山腹中的不明物体。没有人知道它从何而来，也没有人知道它到底是什么。历代接触过它的人，都没有留下完整的记录——不是因为不想，而是因为留下的信息本身就不完整。

## 来源

- **天外来物**，已在此地**数千年**
- 具体坠落年代已不可考——没有人记得是哪个朝代了
- 青山镇形成炼炁士聚居地的根本原因，就是"东西"的存在

## 外观

半透明、不规则的晶体，内部有极慢的光在流转。不发光。不解释。

悬浮在山腹最深处的洞穴中。

## 位置

镇东南方向的山腹深处。经过一条漫长的石道可抵达，石道两侧刻着历代会长的名字。

靠近者需先成为斗炁大会冠军（即会长），才有资格进入。

## 能力

"东西"能够实现**一个作用于许愿者自身的愿望**。

已知机制：

- 许愿后发生的事不一定与许愿者的本意一致——"它说的和你想要的，不一定是同一件事"
- 唯一切实的案例：许愿"回到过去，见那些已经不在的人" → 结果是将许愿者**体穿**至1989年初
- 体穿后记忆保留，但肉身回到过去，需重新经历一生
- 历届来到这里的会长都许了同一个愿望，结果都成为了下一轮的隐藏boss

已知副作用：

- 体穿后无法回到原时间线
- 在陈旧义体化的过程中记忆逐渐丧失（如副会长）

## 对环境的影响

"东西"逸散的能量场影响了整片青山镇的地脉与炁场，使得此地修炼速度是镇外**十倍以上**。

这是炼炁士世代聚居青山镇的根本原因。

（具体机制保持模糊，后续可根据需要补充。）

## 玄门关联

玄门先祖曾向"东西"献祭孪生至亲，获得了御物之力。后立下"双胞胎只能留一个"的规矩试图复现，但从未成功。这一真相只有历代门主知晓。详见 `docs/stories/xuanmen.md`。

## 历史记录

| 时期     | 事件                                                |
| -------- | --------------------------------------------------- |
| 数千年前 | 天外来物坠落青山镇                                  |
| 历朝历代 | 炼炁士逐渐聚居于此                                  |
| 约1989年 | 副会长（当时为会长）接触"东西"，许愿→体穿至1989年初 |
| 约2020年 | 副会长（51岁）与上届会长约定"钟声"后手              |
| 2088年   | 上届会长去世，钟声响起，斗炁大会启动                |
| 2088年底 | 主角成为会长，进入山腹                              |
