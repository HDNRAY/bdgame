# 第 5 章：节点地图与事件系统

## 5.1 地图结构

### 5.1.0 开局剧情三选一

每局游戏开始时，从三个剧情线中选一个。剧情决定第一阶段 Boss 的身份和整局叙事基调：

```
┌─ 选择你的故事 ──────────────────────────┐
│                                          │
│  [1] 心魔                                │
│      「你被过去的阴影所困扰。每一次       │
│       出拳，都像在打自己的倒影。」        │
│      → 第一阶段 Boss: 心魔化身           │
│      → 事件池偏向：内心挣扎/回忆         │
│                                          │
│  [2] 发小对决                            │
│      「那个和你一起长大的兄弟，           │
│       如今站在了擂台的另一端。」          │
│      → 第一阶段 Boss: 童年好友           │
│      → 事件池偏向：羁绊/抉择             │
│                                          │
│  [3] 复仇（假）                          │
│      「有人冒充你的仇人，设下陷阱。       │
│       愤怒驱使你一路追查。」              │
│      → 第一阶段 Boss: 假面仇敌           │
│      → 事件池偏向：追踪/骗局/真相        │
└──────────────────────────────────────────┘
```

> 具体事件文本和分支后续补充。选择不影响数值/机制，只改变叙事 flavor 和第一阶段 Boss 的招式风格。

### 5.1.2 节点分布

```
第一阶段（开局剧情决定 Boss）:
  战斗节点  × 2-3   （路人、野怪）
  事件节点  × 2-3   （剧情相关）
  商店节点  × 1
  锻体节点  × 1
  ─────────────────
  小计 ~8-10 节点
  → Boss: 心魔/发小/复仇（取决于开局选择）

第二阶段（成长过渡）:
  战斗节点  × 2-3
  事件节点  × 2-3
  义体节点  × 1    （第一次接触义体）
  商店节点  × 1
  ─────────────────
  小计 ~8-10 节点
  → Boss: 守门人

第三阶段（大赛）:
  守门人战  →  小组赛第 1 场
  小组赛    →  3 场战斗，每场后 1 个自由节点
  8 强      →  1 场战斗，赛后 1 个自由节点
  4 强      →  1 场战斗，赛后 1 个自由节点
  半决赛    →  1 场战斗，赛后 1 个自由节点
  决赛      →  1 场战斗（终点）
  ─────────────────
  小计 ~16 节点
  → 最终 Boss: 32 强中的胜者

─────────────────────────────────────────
  总计 ~33-36 节点
```

阶段 → Boss          → 最高属性 / 6 维合计
  第一阶段 Boss (10-12 节点):  最高 ~10  / 合计 ~25-35

  第三阶段 小组赛:              最高 ~12  / 合计 ~35-50
        8 强:                   最高 ~15  / 合计 ~45-60
        4 强:                   最高 ~17  / 合计 ~55-70
        半决赛:                 最高 ~18  / 合计 ~60-75

```

### 5.1.3 3 选 1 机制

**全程 3 选 1**，三个阶段都一样。

```

例（完整一局）：
┌─ 开局剧情三选一 ──────────────────────┐
│ [1] 心魔 [2] 发小对决 [3] 复仇（假）│
└────────────────────────────────────────┘
│
▼
┌────────── 第一阶段 ────────────────────┐
│ ○ → 3选1 → ○ → 3选1 → ... → [Boss] │
│ (战斗/事件/商店/锻体) │
└────────────────────────────────────────┘
│
▼ (击败心魔/发小/复仇)
┌────────── 第二阶段 ────────────────────┐
│ ○ → 3选1 → ○ → 3选1 → ... → [守门人]│
│ (战斗/事件/义体/商店/锻体) │
└────────────────────────────────────────┘
│
▼ (击败守门人，晋级 32 强)
┌────────── 第三阶段（大赛） ─────────────┐
│ [小组赛] → 3选1 → [小组赛] → 3选1 → │
│ [小组赛] → 3选1 → [8强] → 3选1 → │
│ [4强] → 3选1 → [半决赛] → 3选1 → │
│ [决赛] │
└────────────────────────────────────────┘

```

**随机池偏向性**（基于角色状态动态调整）：

| 条件         |  战斗   |  事件   |  商店   |  义体   |  锻体   |
| ------------ | :-----: | :-----: | :-----: | :-----: | :-----: |
| 默认         | **40%** | **35%** |   10%   |   8%    |   7%    |
| HP < 40%     |   25%   | **50%** |   10%   |   8%    |   7%    |
| 金币 > 1000  |   30%   |   25%   | **25%** |   10%   |   10%   |
| 已有义体 ≥ 1 |   30%   |   25%   |   8%    | **25%** |   12%   |
| 已有锻体 ≥ 1 |   30%   |   25%   |   8%    |   12%   | **25%** |

商店保底出现：每轮大赛（小组/8强/4强/半决赛）之间至少出现 1 次商店节点。整局约 2-3 次。

**子类型匹配：** 战斗和事件节点内部根据角色状态选择具体内容。

```

战斗子类型选择:
HP < 50% → 低难度战斗（更容易获胜，奖励较少）
已有义体 → 义体对手（击败后概率掉落义体）
已有锻体 → 修炼型对手（击败后概率给锻体相关奖励）
默认 → 随机普通对手

事件子类型选择:
HP < 40% → 治疗/回复类事件优先
已有义体 → 义体相关故事线
已有锻体 → 修炼/奇遇类故事线
默认 → 通用事件池

```

### 5.1.6 32 强对手生成

第三阶段大赛开始时：

```

1. 从 32 位固定选手池中，随机选出 1 位作为"守门人"（已在第二阶段被击败）
2. 剩余 31 人 + 主角 = 32 人，随机打乱生成淘汰赛对阵表
3. 最终 Boss 可能是 32 位中的任意一位（看谁一路赢上来）

```

对手强度方案：

> **按轮次缩放，其余随机**。
> 对手的流派预定义，但具体属性/功法/义体配置完全随机。
> 越到后面的轮次，对手的总属性越高，功法越完整（从 1-2 个杂牌功法逐渐变为完整 build）。

```

轮次 → 对手强度（最高属性 / 6 维合计）
第一阶段 Boss: 最高 ~10 / 合计 ~25-35
第二阶段 守门人: 最高 ~15 / 合计 ~40-55
小组赛: 最高 ~12 / 合计 ~35-50
8 强: 最高 ~15 / 合计 ~45-60
4 强: 最高 ~17 / 合计 ~55-70
半决赛: 最高 ~18 / 合计 ~60-75
决赛: 最高 ~20 / 合计 ~65-80

```

> 守门人作为第二阶段 Boss 已经历过，第三阶段大赛中不会再出现守门人战。

## 5.2 节点类型

### 5.2.1 战斗节点

战斗胜利奖励：

```

固定奖励:
┌─ 金币（第一阶段 30-100，第二阶段 50-200，大赛 200-500）
└─ 概率属性点: 约 30% 战斗掉落属性点 +1（从 2-3 个可选属性中选一）

概率掉落:
第一阶段: 招式 / 普通功法
第二阶段: 招式 / 功法 / 义体
大赛阶段: 功法 / 义体 / 触发器 / 奇物
Boss 战: 必定掉落稀有物品 + 必定属性点 +1

````

> 属性点是稀缺资源，全程专注 1-2 个属性。节点奖励不一定是属性点——可能是招式、金币、功法、回血等。选择奖励时优先补主力属性短板。

战斗类型：

- **路人/野怪** — 第一阶段，无功法，教学级
- **普通对手** — 第二/三阶段，1-2 个功法
- **精英对手** — 大赛中后期，带功法和触发器
- **Boss** — 三个阶段各一个，第一阶段取决于开局剧情选择

### 5.2.2 事件节点

事件是叙事的核心载体。每事件包含描述 + 2-4 选项 + 结果。

```ts
interface GameEvent {
    id: string
    phase: 'phase1' | 'phase2' | 'group' | 'quarter' | 'semi' | 'final'
    title: string
    description: string
    options: EventOption[]
    usedThisRun?: boolean
    weight?: number
    scenario?: 'inner_demon' | 'rival' | 'revenge' // 可选：仅特定剧情线出现
}

interface EventOption {
    text: string
    requirements?: {
        attrs?: Partial<Record<AttrName, number>>
        gold?: number
        hasItem?: string
    }
    result: {
        text: string
        battle?: boolean
        rewards?: EventReward
    }
}

interface EventReward {
    attrs?: Partial<Record<AttrName, number>>
    gold?: number
    hp?: number
    skill?: string[] // 功法
    action?: string[] // 招式
    implant?: string[] // 义体
    trigger?: string[] // 触发器
    artifact?: string[] // 奇物
    forgingLevel?: number // 锻体 +1
}
````

事件选择会触发**属性条件检测**，满足时出现隐藏选项：

```
示例：古墓事件
  "你发现一座被遗忘的古墓..."

  选项 1: "强行打开" → 战斗，随机功法
  选项 2: "解读符文" → 需 insight ≥ 14，insight +1，锻体+1 [隐藏]
  选项 3: "绕道走"  → 无事发生
```

锻体事件示例（替代旧"修炼点"系统）：

```
示例：深山感悟
  "你在深山中遇到一位隐世高人..."

  选项 1: "跟随修炼" → 锻体 +1，回复 30% HP
  选项 2: "切磋请教" → 战斗（赢则锻体 +1 + 招式），输则无事
  选项 3: "婉拒"     → 无事发生
```

### 5.2.3 商店节点

| 商品     |   价格    | 说明              |
| -------- | :-------: | ----------------- |
| 招式卷轴 |  200-600  | 随机 1 个可学招式 |
| 奇物     | 500-3000+ | 随机特殊物品      |
| 疗伤     |  100-300  | 回复 30%-60% HP   |
| 情报     |    200    | 显示下个对手属性  |

> 义体是**独立节点类型**（见 5.2.4），不在商店卖。

### 5.2.4 义体节点

义体节点专门用于改造。每次出现时提供 2-3 件可选购的义体。

```
每件义体包含:
  - 名称和属性加成
  - 价格（500-3000+）
  - 部位和惩罚说明
  - 装备后立即生效（不可退换）

赛前阶段至少出现 1 次义体节点。
```

### 5.2.5 锻体节点

锻体事件直接给 **锻体等级 +1** + 回复 HP。

```
形式多样:
  "跟随师傅修炼"   → 锻体 +1，HP 全满
  "独自闭关"       → 锻体 +1，回复 30% HP
  "深山寻找机缘"   → 锻体 +1 或 属性 +2（随机）
  "击败守护灵兽"   → 战斗，赢则锻体 +1 + 功法

每局游戏约出现 3-5 次锻体节点。
```

### 5.2.6 隐藏 Boss——第 31 人

#### 设计理念

每局大赛本有 32 人。但如果你在赛程中收集齐三块"旧世信物"碎片，第 31 人——**上一次打穿游戏的你自己**——就会出现。

```
正常大赛: 32 强淘汰赛，无第 31 人
触发条件: 赛程中收集 3 块信物碎片（每块在特定事件中取代一个奖励选项）
隐藏 Boss: 上一次你打穿游戏的最终 build
```

#### 信物碎片

三块碎片分布在三个不同事件中，每个事件中有一个选项给你碎片而非常规奖励：

```
事件 A（第一阶段）:
  选项 1: "接受老者的馈赠"     → 获得旧世信物·碎片壹（无属性/金币/物品收益）
  选项 2: "婉拒，要实战经验"   → 战斗，赢则属性点 +1
  选项 3: "索要钱财"           → 获得 150 金

事件 B（小组赛阶段）:
  选项 1: "接过那块残片"       → 获得旧世信物·碎片贰
  选项 2: "换成功法秘籍"       → 随机普通功法
  选项 3: "无视"               → 无事发生

事件 C（8 强阶段）:
  选项 1: "将碎片拼合"         → 获得旧世信物·碎片叁（三块集齐！）
  选项 2: "卖掉碎片"           → 获得 300 金
  选项 3: "扔掉"               → 无事发生

集齐三块 → 下一个节点出现第 31 人
```

#### 战斗

```
隐藏 Boss（第 31 人）:
  强度 ≈ 决赛级别
  配置 = 你上一次通关时的最终 build（属性/功法/绝学/义体/本命物/触发器）
  失败: 正常淘汰
  胜利: 击败"上一次的自己"，此 build 被当前 build 覆盖
```

#### 胜利奖励——旧世遗珍（黑匣子）

获得一个「旧世遗珍」奇物匣，存放在背包中。**在任何准备阶段（赛前/战斗前/节点选择前）**均可打开。打开时根据你当前的 build（已有功法/义体/属性倾向）进行加权随机，**大概率开出对你当前 build 有用的奇物**——比如主堆力量的高概率出力量相关奇物，已有火系功法则高概率出火系强化奇物。

不必在开局就决定，可以等局势明朗后再开。一局只能开一次，开完消失。

#### 与其他隐藏 Boss 的关系

```
此隐藏 Boss 与事件触发的额外战斗不同——第 31 人是大赛的隐藏参赛者，击败他等同于"超越过去的自己"。
```
