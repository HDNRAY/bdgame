# 青山镇设定

> 关联文档：`docs/story.md` — 世界观要素

---

## 行政与规模

青山镇行政上隶属南方某省，名称为"镇"，但实际辖区面积极广。城区、老街区、郊区、山区、林区等多板块并存，整体范围远超一般城镇概念。

人口稠密，其中炼炁士及其后裔占相当比例。

## 区域划分

| 区域                 | 特征                                                                               |
| -------------------- | ---------------------------------------------------------------------------------- |
| **城区**             | 繁华地带，商店、诊所、全息广告牌密集。外来人口与普通居民混居                       |
| **老街区**           | 炼炁士聚居核心，开了几十年的老铺子与新装门头交错。九朵桃花酒吧、宝字堂药铺在此     |
| **镇郊**             | 过渡地带，人烟渐稀。镇口有一台自动售货机，玻璃上有裂纹，里面有几瓶落灰的饮料       |
| **山区（东南方向）** | 荒僻山林，普通人不会深入。东西封藏于此区域的山腹中。禁区标注在地图上，但未写明原因 |

## 炼炁士聚居

青山镇是炼炁士的集中聚居地。原因如下：

- 青山镇的修炼速度是镇外**十倍以上**
- 主因是山腹中"东西"逸散的能量场影响了整片区域的地脉与炁场
- 炼炁士世代聚居于此，并非全凭知识传承，更多是炁本身的牵引——修炼者在镇内自然感受到的进境远超外界，于是留下、扎根、繁衍
- 具体机制无人能说清。可能是"东西"的能量渗入地脉，可能是它改变了此地的天地规则，也可能只是单纯的"待在这里就有用"

> 注：修炼速度差异的具体原因保持模糊，后续可根据需要补充或修改。

## 军方驻扎

军方在青山镇的主力是**特殊作战部队**（俗称斩首部队）。

- 利用此地的高炁环境训练精锐——实战模拟在山中进行，在加速修炼的环境下培养高战力作战人员
- 与炼炁协会为**合作关系**，而非从属关系
- 军方知道青山镇的特殊性（适合修炼），但不知道"东西"的具体细节
- 陆红提是这支部队的教官，掠影为军方斥候

## 禁地

镇东南方向的山，地图上标注为禁区。灵核芯片被带入此地时会触发相关部门处理。

普通人不会去，也不需要知道为什么。

知道的人没有机会说，不知道的人也不会问。
