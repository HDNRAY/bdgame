# 青山镇设定

## 行政与规模

青山镇行政上隶属南方某省，名称为"镇"，但实际辖区面积极广。城区、老街区、郊区、山区、林区等多板块并存，整体范围远超一般城镇概念。

人口稠密，其中炼炁士及其后裔占相当比例。

## 区域划分

| 区域                 | 特征                                                                               |
| -------------------- | ---------------------------------------------------------------------------------- |
| **城区**             | 繁华地带，商店、诊所、全息广告牌密集。外来人口与普通居民混居                       |
| **老街区**           | 炼炁士聚居核心，开了几十年的老铺子与新装门头交错。九朵桃花酒吧、宝字堂药铺在此     |
| **镇郊**             | 过渡地带，人烟渐稀。镇口有一台自动售货机，玻璃上有裂纹，里面有几瓶落灰的饮料       |
| **山区（东南方向）** | 荒僻山林，普通人不会深入。东西封藏于此区域的山腹中。禁区标注在地图上，但未写明原因 |

## 炼炁士聚居

青山镇是炼炁士的集中聚居地。原因如下：

- 青山镇的修炼速度是镇外**十倍以上**
- 主因是山腹中"东西"逸散的能量场影响了整片区域的地脉与炁场
- 炼炁士世代聚居于此，并非全凭知识传承，更多是炁本身的牵引——修炼者在镇内自然感受到的进境远超外界，于是留下、扎根、繁衍
- 具体机制无人能说清。可能是"东西"的能量渗入地脉，可能是它改变了此地的天地规则，也可能只是单纯的"待在这里就有用"

> 注：修炼速度差异的具体原因保持模糊，后续可根据需要补充或修改。

## 时代背景

公元**2088年**。赛博朋克世界观，义体改造技术已经普及。街边的诊所和眼镜店一样多，换一条手臂和配一副隐形眼镜差不多快。全息广告牌在城市上空轮流播放，青山镇也不例外。

炼炁士群体与义体科技共存——有人以纯粹的炁修炼，有人选择用义体强化自身，也有人两者兼修。义体可以延长寿命、增强战力，但代价是长期使用会导致记忆与理智流失。

## 灵核芯片

每个人出生时都会被强制植入一枚灵核芯片，比指甲盖还小。它平时什么也不做，只是安静地待在那里，只在一件事发生时触发相关部门处理：**进入禁区**。

禁区在地图上有标注，但不会写明原因。

## 斗炁大会

会长之位空缺时的继承机制。规则被刻入青山镇地脉：全镇范围内，不分身份、不分流派，以单挑战的方式逐出下任会长。

**流程：** 海选 → 三十二强 → 小组赛（四人一组，积分前二晋级） → 十六人淘汰赛 → 决赛。

海选的最后一关是"守门人"——击败后方可进入三十二强。

## 义体研究部

斗炁协会下属部门。公开目的是研究义体技术与炼炁的兼容性，提升炼炁士的战斗寿命，治疗修炼损伤。

暗面称"**组织**"，实为隐藏boss培养的夺宝势力。详见 `docs/character-relations.md`。

## 军方驻扎

军方在青山镇的主力是**特殊作战部队**（俗称斩首部队）。

- 利用此地的高炁环境训练精锐——实战模拟在山中进行，在加速修炼的环境下培养高战力作战人员
- 与炼炁协会为**合作关系**，而非从属关系
- 军方知道青山镇的特殊性（适合修炼），但不知道"东西"的具体细节
- 陆红提是这支部队的教官，掠影为军方斥候

## 禁地

镇东南方向的山，地图上标注为禁区。灵核芯片被带入此地时会触发相关部门处理。

普通人不会去，也不需要知道为什么。

知道的人没有机会说，不知道的人也不会问。

## 家族与门派

### 药屋

青山镇最古老的家族之一。与御物世家类似，药屋血脉对凝炁诀有着天然的亲和力——这份力量也伴随着代价。每一代药屋后人，或早或晚，先天或后天，必失五感之一。

药屋族人往往在失去某一感官后，反而走向武学或医术的巅峰：失去的感官被炁感知所补偿。

| 名字 | 失去 | 备注               |
| ---- | ---- | ------------------ |
| 黛玄 | 听觉 | 特殊事件调查科     |
| 小花 | 视觉 | 拳掌宗师，无明之明 |
| 来风 | 味觉 | 空拳               |

**家传功法：凝炁诀** — 药屋家传呼吸法，只限血脉传承。从小修习，以炁贯通全身。与御物家族的体质限定相似，外人无法修习。

### 玄门（御物世家）

御物世家，以炁御器。先祖曾向"东西"献祭血亲，获得了强大的御物之力。这股力量通过血脉传承至今。

玄门有一条历代传下的规矩——**双胞胎只能留一个**。后世多次试图复现先祖的献祭以获取更强之力，但从未成功。规矩却一代代传了下来，只有玄门门主才知其真正的源头。

玄门线独立于组织体系，是纯粹的家族故事。

### 归海楼

青山镇的武学门派，以居合剑术闻名。广交天下豪杰，定期举办比武大会。

- **掌门一刀** — 居合剑术宗师
- **弟子桑原** — 空手道高手，以血祭之术闻名
- **弟子橘子真** — 三节枪使，枪术惊世却无争强之心

### 天工坊

天工坊主人**千星**，陨铁神珍、千机、阿赖耶识的锻造者。

每人一生只能给他下一个订单，做不做全凭他意。

斗炁方式为以炁驱动的电磁锤纵横，雷火交加。

### 斗炁图书馆

位于街角的一座古朴图书馆，檐下匾额写着「斗炁图书馆」四个字。

- **管理员龙语仙** — 龙角少女，擅长防御功法
- **常客白狐·南宫** — 冰霜双刀使，埋头翻阅攻击功法古籍
