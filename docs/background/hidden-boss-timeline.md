## 核心设定

- **身份**：上一轮选择了"回到过去"的主角。斗炁协会副会长，120岁
- **底色**：他从来不是坏人。他的悲剧在于：用做好事的初衷，一步步走向了自己不想成为的样子
- **堕落机制**：义体越来越多→有机体越来越少→"东西"的残留能量趁机侵蚀→记忆和意识层层剥落→只剩执念空壳
- **最后的良心**：倒下时最后一瞬清醒浮上来，无声说"别信它"

---

## 组织创立初衷

副会长创立义体研究部（及"组织"），真实目的有二：

1. **培养战力** → 帮他赢得斗炁大会 → 再次成为会长 → 面对"东西"
2. 如果自己撑不到2088年，让培养的人替他进山

他没有对任何人提过"东西"的存在。组织成员知道的是：

> **"研究义体技术与炼炁的深度结合，打造新一代义体炼炁士——不受协会规章限制。"**

这是青山镇正当且前沿的方向。成员们以为在推动义体科技的边界，不知道真正目标在山腹里。

学校、义体研究部、组织三者关系：

- **学校**：独立运作，纯粹收容教育孤儿。**不知道组织的存在**，最多知道老大是同一个人（副会长）
- **义体研究部**：协会公开部门，表面研究义体，大部分成员不知道暗面
- **组织**：义体研究部的内圈，知道部门有"更激进的目标"，但只有军师和博士知道全貌

## 时间线

### 1989年 · 体穿抵达

- 他原本就是青山镇本地人，从小就知道"在这里修炼比外面快"，但一直以为是因为祖传功法的缘故
- 20岁出头成为会长，进入山腹，第一次面对"东西"
- 许愿"再见故人"，下一秒站在1989年的青山镇街头
- 此时他终于明白：**修炼快的真正原因是这个鬼东西**
- 最初的念头：**"东西"在愚弄所有人——历届会长都被它编进循环了，我要终结它**
- "东西"的力量在他身上有残留，他能感觉到，但以为是许愿后劲

### 1990s · 蛰伏与新发现

- 生活多年后确认了一件事：青山镇的修炼速度比自己小时候还快——说明"东西"的能量是在波动的，不是一个恒定值
- 开始系统性地研究"东西"的规律——查阅历任会长笔记、走访老人
- 越研究越发现：这东西不是"能许愿"那么简单，它在以某种方式影响整个镇子
- 此时目标清晰：**变强、活到重新成为会长、进山、毁了它**

### 2000s · 创办青山第二寄宿学校

- 镇上有孤儿流落——他认识其中一些孩子的父母，因炼炁士争斗去世
- 他决定办一所学校，给无家可归的孩子一个去处
- **初衷是好的**——不收学费，教修炼、教识字、给出路。跟穿越无关，他有能力就会做
- 这所学校后来成了组织招募的渠道之一，但那不是他办校的本意

### 2010s · 结识上届会长，共同治镇

- 上届会长（年轻人，刚继位）主动找上他——听说过他的实力和名声
- 两人成了挚友。他透露了一些关于"东西"和体穿的信息（保留最关键部分）
- 一起处理镇上的纠纷：炼炁士斗殴、外来势力渗透、普通人误入禁区
- **此时他是青山镇的隐形守护者**——会长主外，他主内

### 2020年 · 钟声之约（51岁）

- 两人密定后手：会长如果非正常死亡，钟声强制触发斗炁大会
- 目的：**确保每一代都有够强的人拥有面对"东西"的资格**
- 同时开始想：如果自己撑不到2088年，需要有接班人替他进山
- 第一批培养对象：张烈那一代（十岁出头的孩子）

### 2020s · 第一批义体

- 50多岁，自然衰老拖累战力。第一次加装：**左臂**
- 小手术，几乎没影响，意识百分百清醒
- "东西"的残留能量仍在，但没什么异常

### 2030s · 创立义体研究部

- 为储备长效战斗能力，在协会内部设立义体研究部——表面是正规科研部门
- 招募博士——博士那时年轻、热情，真心对义体技术痴迷，不知道背后深层目的
- **此时他仍然清醒，仍然记得钟声的意义，仍然以"摧毁东西"为目标**

### 2040s · 第二次义体

- 换**右臂、双腿关节**——手术更大，恢复期更长
- 偶尔出现记忆模糊：忘了昨天跟谁说过话。以为是年纪大了，没在意
- "东西"的残留能量**开始悄悄放大这种模糊**

### 2050s · 大面积义体化开端

- 换**胸腔外壳、脊柱**——全身约60%义体化
- 记忆断片变得频繁。开始把组织事务写成**手账**，靠笔记过日子
- 隐约记得"要走到山腹"这个终点，但通往终点的路——钟声、学校、摧毁东西——**开始模糊**

### 2060s-2070s · 意识滑坡加速

- 换**下半张脸、部分躯干**——除大脑外几乎都在换
- 记忆大面积丧失。**手账出现重复条目、自相矛盾的指令**
- "东西"的残留能量在此阶段作用最明显——**它在主动模糊他对"为什么"的记忆，只保留"要拿到"**
- 不再直接管理组织——不是想放权，而是已经理不清了

### 第一代军师 · 义体研究部副部长（约2065-约2080）

- 在副会长意识模糊时期接手组织运行
- 比他小30-40岁，是学校培养的第一批精英
- 知道部分计划（"要拿到东西"），但不知道钟声细节和"东西"全貌
- **作风激进**，将组织目的扭曲为"不惜一切代价扩张义体研究"
- **2074年林家反对义体研究，他擅自下令灭门**——此时副会长已意识模糊，不知此事
- 灭门没留下痕迹，但组织扩张太快，**引起军方注意，开始派卧底**（张烈）
- 组织内部不满积累——老人认为灭门越过了"不制造孤儿"的底线
- **约2080年，被当时十几岁的现任军师策划推翻**——义体需定期注射维护针剂，那天博士"没空"，让副部长自己操作。军师提前把针剂调了包，副部长"拿错了"，诱发致命排斥。表面是操作失误，实则精心设计
- 这是一场沉默的政变：无人动手，但关键人物（博士、第一代老人）事后都选择了沉默
- 死后由现任军师接管，组织恢复收敛作风

### 2080s · 第二代军师 = 现在的军师

- 天生道种线的师兄（比主角大2岁），**9岁时宗门假死**——被过渡者选中带走
- 过渡者看中他的天赋，觉得他像自己，想把他培养成接班人
- 但过渡者**误判了**——军师比他更高一筹。他在组织里观察了几年，看透了过渡者的疯狂
- 十几岁时策划了过渡者的"意外"——不是反噬，而是一个天才在清醒状态下做出的选择
- 接手组织后恢复收敛作风，不制造孤儿、不滥杀。他确实没有普世道德观念——但也不认同过渡者的做法，只是觉得那样"太蠢了"
- 约2082-2085年间，13-16岁的他以"军师继任者"身份接手组织

### 2088年 · 终局

- 会长去世，钟声响起
- 副会长听到钟声，已不记得含义。但他知道：**有人会来。他要去山腹。**
- 站起来，走向山腹。挡住主角的路。战斗
- 倒下时，义体外壳碎裂。**最后一瞬清醒浮上来**——他看着主角，口型："别信它"
- 那是真正的他，被压在百年义体之下、最后一小块没被"东西"吞掉的自己

## 关键设计要点

| 元素                   | 设定                                                                                                         |
| ---------------------- | ------------------------------------------------------------------------------------------------------------ |
| **他是不是坏人**       | 从来不是。他的悲剧在于：每一步都是为了好的目的，但累积的结果不可逆                                           |
| **转折机制**           | 不是单一事件，是每次换义体都丢一点，几十年累积成不可逆的滑坡                                                 |
| **"东西"的作用**       | 许愿时残留的力量一直在暗处侵蚀他。义体越多→有机体越少→"东西"越容易渗透。它需要一个记忆模糊、只剩执念的守卫者 |
| **学校为什么一直办**   | 这是他意识深处最后一道干净的东西——收留孤儿这件事不需要"记得"，是他本能的善良                                 |
| **会长知不知道他变了** | 知道一些（感觉到他变了），但不知道"东西"在背后起作用，只以为是义体的代价                                     |
